# Serverless Task Tracker App

## Project Overview
This project is a serverless web application built using AWS services such as
S3, API Gateway, Lambda, DynamoDB, and Cognito.

## Architecture
User → S3 → Cognito → API Gateway → Lambda → DynamoDB

## Features
- User authentication using Cognito
- Task CRUD operations
- Fully serverless architecture

## Technologies Used
- Amazon S3
- AWS Lambda
- Amazon API Gateway
- DynamoDB
- AWS Cognito
